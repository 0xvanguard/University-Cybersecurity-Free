## 📋 Descripción

Profesional que **valida controles de seguridad** contra estándares (ISO 27001, SOC 2, PCI-DSS, NIST 800-53, sectoriales). Modalidades:

- **Auditor Interno** — empleado de la empresa (Internal Audit dept.)
- **Auditor Externo / Tercera parte** — Big 4, certificadores (BSI, BV, SGS, AENOR)
- **IT Auditor (CISA)** — auditoría de sistemas de información
- **Auditor Forense** *(senior)* — fraude + cyber

Diferencia con `iso-27001-lead-implementer`:
- **Implementer construye** el SGSI
- **Auditor verifica** que el SGSI cumple

Carrera estable, predecible, **buena puerta de entrada** para perfiles administrativos / contables hacia ciberseguridad.

---

## 👤 Roles típicos

- **IT Auditor (Internal)** — banca, retail, telco
- **External Auditor (Big 4)** — Deloitte, EY, PwC, KPMG
- **Certification Auditor** — BSI, Bureau Veritas, SGS, AENOR (ISO certs)
- **SOC 2 Auditor** — typical para SaaS US
- **PCI-QSA** — Payment Card Industry Qualified Security Assessor
- **Audit Manager / Director** *(senior)* — coordinación de equipos

## 🗺️ Roadmap (12 meses)

### Fase 1 — Fundamentos (0-3 meses)
- [ ] ISO 27001:2022 + ISO 19011 (auditoría de sistemas) leídos
- [ ] COSO + COBIT 2019 base
- [ ] Diferencia: Tipo I vs Tipo II audits (SOC 2)
- [ ] CISA empezado

### Fase 2 — Práctica (3-6 meses)
- [ ] Evidence collection: 20+ controles ISO mapeados a evidencias
- [ ] Mock audit interno completo (test of design + test of operation)
- [ ] Sampling estadístico aplicado a una población de cambios
- [ ] **CISA aprobado** 🎯

### Fase 3 — Profesional (6-12 meses)
- [ ] Acompañar audit real (interno o externo, Big 4 o boutique)
- [ ] Plan anual de auditoría basado en riesgo
- [ ] ISO 27001 Lead Auditor empezado
- [ ] Especializarse: SOC 2 / PCI / ISO / IT general
