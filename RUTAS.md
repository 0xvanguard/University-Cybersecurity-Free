### Tramo final: MLSecOps y Seguridad de LLM

- **Módulo:** MLSecOps y seguridad de LLM (03-IA-AGENTES-HERRAMIENTAS/03-mlsecops-llm-security/)
- **Niveles:** Entry → Intermedio → Avanzado.
- **Duración estimada:** 4–6 semanas según dedicación.
- **Requisitos:**
  - Haber completado la Ruta de Fundamentos.
  - Conocimientos básicos de Python (opcional pero recomendado).

En este tramo aprenderás a:

- Entender qué es un LLM y qué implica MLSecOps.
- Diseñar y ejecutar pruebas básicas de seguridad para modelos de lenguaje.
- Proponer un pipeline sencillo con controles de seguridad para IA.
- Generar tres tipos de entregables públicos: post educativo, repo de pruebas y guía/diagrama de pipeline.
