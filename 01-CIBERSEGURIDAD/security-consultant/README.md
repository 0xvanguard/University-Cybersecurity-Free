## 📋 Descripción

Rol **client-facing** que combina experticia técnica + comunicación + business sense. Trabaja con clientes externos (consultoría) o áreas internas (advisory) en proyectos definidos: assessments, gap analysis, implementación de marcos, vCISO, training, due-diligence M&A.

Cuatro modalidades principales:
- **Big 4 / Tier 1** (Deloitte, EY, PwC, KPMG, Accenture) — entry-level abundante, exit hacia industria
- **Boutiques especializadas** — pentest, IR, OT (Mandiant, Coalfire, NCC Group)
- **MSSPs / MDR** — Arctic Wolf, Optiv, NTT
- **Independiente / freelance** — vCISO, advisory part-time

---

## 👤 Roles típicos

- **Security Consultant (Big 4)** — assessments, control reviews, audit prep
- **vCISO / Fractional CISO** — varias empresas en paralelo
- **Pentest Consultant** — boutiques ofensivas
- **Cloud Security Consultant** — migraciones + revisiones
- **Privacy Consultant** — GDPR / Ley 1581 / HIPAA implementations
- **M&A Cyber Due Diligence** — pre-acquisition assessments

## 🗺️ Roadmap (12 meses)

### Fase 1 — Fundamentos (0-3 meses)
- [ ] Inglés escrito B2+ (oral B1 mínimo) — **non-negociable**
- [ ] NIST CSF + ISO 27001 + SOC 2 entendidos a nivel mapping
- [ ] Soft skills: presentación de 15 min sobre un tema técnico
- [ ] Security+ aprobado

### Fase 2 — Especialización vertical (3-6 meses)
- [ ] Elegir nicho: GRC | Cloud | Pentest | Privacy | IR
- [ ] Cert del nicho (CISA / AWS Sec Spec / OSCP / CIPP)
- [ ] Reporte modelo: gap analysis ficticio formato Big 4
- [ ] Networking: 1 evento mensual + LinkedIn activo

### Fase 3 — Profesional (6-12 meses)
- [ ] Primer engagement (real / pasantía / Big 4)
- [ ] CISSP empezado
- [ ] Construir personal brand (blog / talks / posts técnicos)
- [ ] Plantillas reutilizables: 5 templates de assessment
